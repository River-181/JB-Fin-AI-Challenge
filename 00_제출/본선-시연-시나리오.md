## 0. 시연 시나리오 선정 근거

| 항목 | 결정 |
|------|------|
| **히어로 시나리오** | **GP-3: 전주 중앙로 카페 소상공인 (SME) end-to-end** (`?demo=sme`, 케이스 `JBG-104`) |
| **보조 시나리오** | **거버넌스·PII 구간** (히어로 내 포함) + **GP-2 보이스피싱 차단** (60초 압축) |
| **제외** | GP-1 전세 보호 — 히어로 충분 커버 후 시간 잔여 시에만 언급 |

**선정 이유:**
- SME 시나리오는 판단(상환위험 분류) → 행동(정책금융 매칭·콜백 초안) → 거버넌스(토큰화·반출 스캔) → 승인 게이트 → 감사 원장 상태변화를 **단일 경로 안에서 모두 시각화**할 수 있다.
- 심사 기준 2.4(개인정보·규제)·3.1(판단·행동·검증 구조)·5.5(리스크 대응)의 핵심 차별점을 **거버넌스·PII 구간**이 정면으로 커버한다.

## 1. 오프라인 구동 체크리스트

> 본선 현장은 네트워크·외부 컴퓨팅이 제공되지 않는다. **본인 장비(MacBook) + 로컬 서버로 완전 오프라인 구동.**

### 1-A. 사전 준비 (전날 또는 출발 전)

```bash
# 1. 앱 경로 확인
ls /Users/river/project/active/JBproject/02_제품/app/
# → app.js  index.html  modules.js  styles.css 존재 확인

# 2. python3 설치 확인 (macOS 기본 포함)
python3 --version

# 3. 브라우저 캐시 프리로드 (오프라인 대비)
#    Chrome/Safari에서 아래 URL을 미리 방문해 캐시 적재
#    http://127.0.0.1:8011/index.html
#    http://127.0.0.1:8011/index.html?demo=sme
#    http://127.0.0.1:8011/index.html?demo=phishing

# 4. (선택) 배터리 절전 모드 해제, 화면 자동잠금 끄기
#    시스템 설정 → 배터리 → 전원 연결 중 절전 끄기
#    시스템 설정 → 잠금 화면 → 화면 꺼짐 Never
```

### 1-B. 현장 기동 명령 (발표 10분 전)

```bash
# 앱 디렉토리로 이동 후 로컬 서버 기동
cd /Users/river/project/active/JBproject/02_제품/app
python3 -m http.server 8011

# 브라우저에서 확인
# → http://127.0.0.1:8011/index.html
```

### 1-C. 구동 확인 체크리스트

- [ ] 터미널에 `Serving HTTP on 0.0.0.0 port 8011` 출력 확인
- [ ] 브라우저 주소창: `http://127.0.0.1:8011/index.html` → 대시보드 렌더링 확인
- [ ] `?demo=sme` 파라미터 시 "GP-3 소상공인 자금압박" 코치 배너 출현 확인
- [ ] `?demo=phishing` 파라미터 시 "GP-2 보이스피싱 차단" 코치 배너 출현 확인
- [ ] 브라우저 확대율 100% (또는 발표 화면 해상도에 맞게 조정)
- [ ] 발표 화면 미러링/프로젝터 연결 확인
- [ ] 사전 녹화 영상 파일 위치 확인 (폴백용): `00_제출/시연영상/`
