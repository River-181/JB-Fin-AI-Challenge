## 핵심 차별점 1문장 (반복 강조)
"외부 프런티어 LLM의 추론력은 활용하되, 고객 원본 PII·신용정보는 절대 외부로 반출하지 않는다 — 신용정보법 §40조의2·개인정보보호법 §28조의4/§28조의5에 정합하는 4중 방어로, 금융권 실도입이 가능한 운영체계."

- "실데이터 없는데 신뢰 가능?" → 출처 등급 칩(공공/시뮬레이션/추정) 구분, 공공 API 인벤토리·라이선스 정리됨(기능명세 §6-A).
- "외부 LLM 쓰면 규제 위반 아닌가?" → 망분리 개선 로드맵(2024-08-13, 다층보안체계)으로 생성형 AI 이용 허용 흐름 + PII 토큰화·국내 라우팅으로 원본 비반출.
- "JB와 실제 연관?" → 2025-12-26 네이버클라우드 MOU(상담→심사→사후관리 AI) 공표 방향의 실행 수단(운영 중 주장 아님).
- "환각/책임소재?" → 에이전트는 근거·불확실성·다음 확인 항목 의무 출력, 승인 게이트+감사 원장으로 책임소재 추적(06_LLM위키/03).
