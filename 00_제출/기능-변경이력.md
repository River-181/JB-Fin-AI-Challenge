## 예선 이후(본선) 예정

> 아래는 커밋된 기능이 아니라 [`_canon.md`](../_canon.md)·기능명세서 "향후 발전 방향"에 명시된 계획이다.

- 공공 데이터 API 실연동(공공데이터포털·한국은행 ECOS·HUG·국토부 실거래가), 등기/실거래는 사람 트리거+캐시로 대량수집 제약 대응.
- 국내·온프레 모델(원본 PII)과 외부 프런티어 모델(토큰 입력) 라우팅 실연동, RAG(공공/내부 문서) + Rule Engine(승인 레벨) 결합.
- 정적 함수 계약(`computeRiskDecision`·`buildDashboardData`·`auditChainRecords`·`moveCaseToColumn`)을 서버 API로 1:1 승격.
- 위험 점수 산식 고도화, 실행 실패 복구(재시도·SLA 상위보고), 계열사·업무영역·고객군 확장.
