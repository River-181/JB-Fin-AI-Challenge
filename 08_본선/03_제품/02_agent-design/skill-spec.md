# 스킬 명세

> 역엔지니어링/브레인스토밍으로 채울 예정

---

## 스킬 설계 원칙

> 각 에이전트는 스킬(Tool)을 장착해 특정 기능을 수행한다. 스킬은 독립적이고 재사용 가능해야 한다.

---

## 씨앗 포인트

- **씨앗**: 에이전트 상세의 Skills 탭에서 각 에이전트가 장착한 스킬 목록·설명·입출력 형식을 확인할 수 있음
- **씨앗**: 스킬은 RAG 기반 금융 규정 검색, 고객 데이터 조회, 메시지 초안 생성 등으로 구성
- **씨앗**: 각 스킬 실행은 AuditEvent로 기록되어 감사 추적 가능
