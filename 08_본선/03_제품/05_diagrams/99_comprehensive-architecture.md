# 종합 아키텍처 — 심사용 시스템 구성도

> 역엔지니어링/브레인스토밍으로 채울 예정
>
> **빠른 진입**: [[본선 HOME|본선 HOME]] → 빠른 진입 섹션

---

## 목적

심사위원이 시스템 전체를 단 하나의 다이어그램으로 이해할 수 있는 종합 구성도. 모든 레이어를 포함.

---

## 씨앗 포인트

- **씨앗**: 다크 3열 쉘 (org-rail + sidebar + workbench + 속성패널) — UI 레이어
- **씨앗**: 오케스트레이터 → 4개 서브에이전트 → 스킬(RAG·규칙 엔진·DB 조회) — 에이전트 레이어
- **씨앗**: Case/AgentRun/Approval/AuditEvent — 데이터 레이어
- **씨앗**: 승인 게이트 — Human-in-the-Loop 레이어

---

## 종합 다이어그램

> 작성 예정 — Excalidraw 또는 Mermaid로 작성
>
> Excalidraw 원본: `assets/excalidraw/comprehensive-architecture.excalidraw`

```
┌─────────────────────────────────────────────────────────────┐
│                    JB LocalGuard OS                         │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │               UI 레이어 (다크 3열 쉘)                │   │
│  │  [org-rail] [sidebar: 케이스 목록] [workbench]       │   │
│  │                        [속성 패널]                   │   │
│  └──────────────────────┬──────────────────────────────┘   │
│                         │                                   │
│  ┌──────────────────────▼──────────────────────────────┐   │
│  │             에이전트 레이어                          │   │
│  │  [오케스트레이터]                                    │   │
│  │    ├─ [Triage Agent]     스킬: 분류                  │   │
│  │    ├─ [Risk Assessor]    스킬: 위험도 평가            │   │
│  │    ├─ [Draft Writer]     스킬: 초안 생성              │   │
│  │    └─ [Compliance Chk]  스킬: RAG 규정 검색          │   │

│                         │                                   │
│  ┌──────────────────────▼──────────────────────────────┐   │

│  │  담당자/감독자 승인 없으면 → 외부 발송 불가           │   │
│  └──────────────────────┬──────────────────────────────┘   │
│                         │                                   │
│  ┌──────────────────────▼──────────────────────────────┐   │
│  │              데이터 레이어                           │   │
│  │  Case | AgentRun | Approval | AuditEvent            │   │
```
