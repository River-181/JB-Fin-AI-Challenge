# 쇼케이스 빌드플랜 — 풀 쇼케이스(10흐름 × 6페르소나) 도장깨기

> **[대외비]** 본선 시연·영상·발표용 내부 빌드 계획. 인물·거래는 전부 합성(PII無). 외부 공개 전 사람 승인.
> **목적**: 승보 프로젝트2 뷰를 최소 기준으로 두고, **실제 작동하는 흐름 몇 가지**를 선정해 시나리오·시연영상·발표로 1등을 노린다. **최대 풀 쇼케이스**로 잡고 **하나씩 도장깨기**(가능하면 전부).
> **히어로(2026-07-04 확정)**: 딥 실LLM 히어로 = **P1 김건우(전주 중앙로 카페, CCL-0001) 유지**(폐기 아님, PR#1 기배선). Figma `Problem Definition`(node 4815-9181)의 쇼케이스는 **6페르소나 확장**(김건우+오태림 JBWC·문서희·윤복례·임세빈·전세, 송민재 백업). 2계열사=전북은행+JB우리캐피탈. 이 계획이 반영 SSOT다.

## 3. 실 LLM은 어디에 (확정)

**F1 딥 실 LLM = P1 김건우(CCL-0001, 전북은행 기업여신).** PR#1(`feat/live-agent-ccl-financial`)이 **이미 CCL-0001에 배선**돼 있어 재시드 없이 최단 경로. 텍스트 생성(품의/회신 초안)이 LLM 가치가 선명하고, **Zero-PII**(원문 대신 최소필드만 외부, PII는 로컬모델 처리) + **검증 에이전트가 자기 오류를 잡는 장면**이 신뢰의 절정.
- **오태림(P2)** = 별도 **JBWC 기업여신 흐름**(기술신용 근거 패키징) — 실 LLM 2번째 후보 or 목업. F1-오태림 설계도는 **JBWC 흐름으로 재프레이밍**(딥 히어로는 김건우로 이전).
- 실 API 근거: F2 문서희(기상특보)·**P6 전세(전세 실거래가 `?live=1`, 유일 검증된 실 fetch)**.

## 3-1. 런타임 — 모델 선택 게이트웨이 (결정 4·6)

**어드민이 모델을 고르는 게이트웨이.** `/llm` 게이트웨이를 **모델 선택기**로 확장: **claude code CLI / codex CLI / 로컬모델**(EXAONE·Qwen 등) 중 어드민 계정에서 선택.
- **PII 하이브리드**(결정 6): **로컬모델이 PII 포함 처리**(원문 반출 없이 온프레), **프런티어(claude/codex)는 비민감·최소필드만**. → "PII 비반출"의 이중 방어(로컬 처리 + 필드 최소화). **로컬 불가 시 codex 폴백**(정직 표기).
- Gate 0(구 "이식")= 이 **모델 선택 게이트웨이를 JB_project2에 이식** + PR#1 머지. 예선 `02_제품`의 `/llm`(handleLlm·llm-runs.jsonl) 로직 재사용.
- 폴백 사다리(선택 모델 실패→다음 모델→사람 큐 `escalated`) 유지, 모든 시도 `llm-runs.jsonl` 기록.

## 4. 도장깨기 빌드 순서 (의존성 기반)

> 각 라운드 = 커밋 + 시연 가능한 슬라이스. 라운드 끝마다 데모 리허설 가능.

- **Gate 0 (선행·개발 게이트)**: **모델 선택 게이트웨이(claude/codex/로컬)를 JB_project2에 이식** — 현재 예선 `02_제품`(handleLlm·llm-runs.jsonl·liveLlmBlock)에만 있음. 제출 정본=JB_project2에서 실 LLM(F1)을 보이려면 이식 필수. + PR#1(`ccl-financial`, CCL-0001) 머지.
- **Round 1 — 코어(최소 세트, "이게 진짜 도냐" 격파)**: **F1**(실 LLM 김건우 CCL-0001) → **F2**(실 API 문서희 기상) → **F6**(실 API 전세) → **F7**(계기판).
- **Round 2 — 차별성("왜 너희냐" 격파)**: **F8**(나선형/조직도) → **F9**(엔터퍼스트) → **F3**(오태림 JBWC 확장성).
- **Round 3 — 풀 쇼케이스**: **F4**(피싱 윤복례) → **F5**(임세빈 학생) → **F10**(119/격리) → 백업 F3′(송민재).

**시간 리스크 폴백**: 예선 5분/결선 15분. Round 1(4흐름·실동작 3종)이 최소 승부 세트. Round 2까지가 1등 타깃. Round 3은 부록/백업 슬라이드·시연영상으로.
