# RAG·규칙 엔진

> 역엔지니어링/브레인스토밍으로 채울 예정

---

## 역할

> 에이전트가 금융 규정·사내 규칙·판례를 검색해 근거 있는 판단을 내리도록 지원.

---

## 씨앗 포인트

- **씨앗**: Compliance Checker 에이전트의 핵심 스킬 — `규정_검색 (RAG)` SK-04
- **씨앗**: 전세사기 관련 법령·시행령·판례 검색 → 위험도 평가 근거 제공
- **씨앗**: 규칙 엔진 (Rule Engine) 은 하드코딩된 금융 규정 임계값 처리 (RAG가 커버 못하는 정량 규칙)

---

## RAG 파이프라인 (작성 예정)

```
질의 입력
  → 임베딩 (Embedding)

  → 컨텍스트 주입
  → LLM 생성
  → 출처 인용 반환
```
