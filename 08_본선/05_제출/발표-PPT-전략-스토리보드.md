# 발표 PPT 전략 및 스토리보드

> 본선 발표 흐름은 [[회의록-2026-07-03-제품정의확정]]·[[회의록-2026-07-04-시연기술범위]] 이후 확정한다. 이 문서는 확정 전까지 유지하는 **PPT 전략 SSOT**이며, 최종 덱은 여기서 잠금 항목만 가져간다.

## 현재 전략

| 항목 | 기준 |
|---|---|
| 발표 목표 | 제품 완성도 + JB 업무 적합성 + "AI와 협업해 문제를 푸는 팀"을 동시에 증명 |
| 핵심 문장 | JB LocalGuard OS는 RM 현장의 위험 신호를 케이스로 묶고, AI 에이전트가 판단·행동 초안을 만들되, 고객 대상 행동은 사람 승인 전까지 막는 금융 AI 운영 콘솔이다 |
| 발표 톤 | 화려한 기술 과시보다, 금융권이 바로 켤 수 있는 통제·근거·책임 구조를 전면화 |
| 시연 우선순위 | SME 히어로 1개를 완주하고, 보이스피싱·전세는 시간 잔여 또는 백업 슬라이드 |
| 미확정 | 발표 시간, 최종 히어로 도메인, 실API/로컬/정적 범위, 본선 현장 추가 구현분 |
