## 쓰는 법
1. 각 `D*.md` = 복붙용 딥리서치 프롬프트. 해당 엔진(GPT/Gemini)에 붙여넣어 실행.
2. `_결과/` = 회수된 리포트(docx→md, 워드 수식은 LaTeX 변환). `_종합-*` = 출처 적대검증본.
3. 더 돌릴 땐 프롬프트 복사 → 딥리서치 → 결과 공유. 발표·제품설계 근거로 활용.
