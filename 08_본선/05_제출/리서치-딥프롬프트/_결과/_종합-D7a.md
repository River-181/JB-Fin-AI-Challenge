## 핵심 발견

**즉시 투입 가능 축(공개형 API):** 국토부 RTMS 실거래가 계열(6종), 국세청 사업자 상태조회, 소상공인시장진흥공단 상권정보, 행정안전부 인허가(통신판매업 포함), HUG 전세보증·사고 파일데이터(3종), 금융감독원 OpenDART, 금융위원회 금융통계국내은행정보. 모두 data.go.kr 자동승인·이용허락범위 제한 없음·무료.

**유료·건별 조회 축:** 인터넷등기소 부동산 등기 열람(700원/통)·발급(1,000원/통). 개별 등기사항증명서 본문을 대량 자동취득할 공식 Open API 경로는 양 run 모두 미확인. 전세 권리분석의 마지막 1km는 이 경로가 유일하며, 자동화 허용 여부는 [미검증].

**기관계약형 축:** 금융결제원 오픈뱅킹·어카운트인포·FIN MAP. 승인·계약·적합성심사 필요, 수수료 존재(금액 비공개). MVP가 아닌 PoC 이후 제휴 과제로 분리.

**경로 변경 주의:** localdata.go.kr 개발센터는 2026-04-16 폐쇄. 지방행정 인허가 API는 data.go.kr 표준경로로만 사용.

**run 간 핵심 불일치(확인 완료):**
1. 등기정보광장 API 키 유효기간: Run1 = "1년" → Run2 = "24개월(3개월 미사용 자동삭제)" → **WebSearch 합의: 24개월이 정확. Run1 오기.**
2. ECOS 호출 한도: 양 run 모두 [미검증]. WebSearch도 공식 수치 미확인 → **[미검증] 유지.**

## 제품 적용 — 도메인별 데이터 매핑

### 소상공인 (RM 리스크 피처)

| 데이터 | 역할 | 조합 방식 |
|---|---|---|
| 국세청 사업자 상태조회 (#7) | 휴·폐업 플래그, 과세유형 | 사업자등록번호 입력 → 상태 플래그만 보관 |
| 소상공인시장진흥공단 상권정보 (#8) | 입지밀도, 동종업종 경쟁도 | 주소 기반 반경/행정경계 조회 |
| 행안부 인허가(일반음식점·통신판매업) (#9, #10) | 인허가 정상영업 여부 | 업종코드 매칭 |
| 한국은행 ECOS (#11) | 금리·물가·경기 민감도 | 거시지표 배치 적재 |
| 금융위 금융통계 국내은행정보 (#19) | 은행권 대출·예수금 매크로 | 지역별 집계 인풋 |

**주의:** 상권정보 API는 메타 실시간이지만 표준데이터 배치 분기. 실시간/배치 시점 분리 필요. 사업자등록번호 원문은 외부 LLM 미전송.
