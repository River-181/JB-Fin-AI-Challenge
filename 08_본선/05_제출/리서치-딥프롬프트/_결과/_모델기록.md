# 리서치 모델 기록 (Model Attribution)

> ⚠️ 대외비. 나중에 "어떤 모델로 리서치했나" 질문 대비. 딥리서치 산출물별 사용 모델·기능.

## 방법론 (사용자 정본)
- 🔵 **Gemini** — **Gemini Flash / Pro** + **Deep Research** 기능.
- 🟠 **ChatGPT** — **GPT‑5.5**, **high thinking** + **Deep Research**.
- 🟣 **Perplexity** — **Deep Research**(초기 일부 도메인, 한도 소진 후 GPT/Gemini로 재배정).
- 🟢 **Codex** — `gpt‑5.3‑codex‑spark`(초기, 긴 웹 딥리서치서 컨텍스트 오버플로 실패) → **`gpt‑5.5`(CLI 0.142.4)**. **웹 딥리서치가 아니라** 로컬 점검·프롬프트 작성·갭감사·결과 적대검증 종합에 사용.
- 도구(Claude/Sonnet/Opus): 오케스트레이션·프롬프트 엔지니어링·적대적 출처검증(format B 종합).
