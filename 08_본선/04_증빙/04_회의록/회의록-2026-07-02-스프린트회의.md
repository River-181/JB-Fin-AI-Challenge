# 회의록-2026-07-02-스프린트회의

> JB금융그룹 Fin:AI Challenge **본선 스프린트 회의**(7/2 밤, 온라인). 원문(대외비)=[[원문-2026-07-02-스프린트회의-JB발췌]].
> ⚠️ 화자 실명 STT 추정. STT 수치(결제·대출·페이지수 등)는 **미검증** → `_canon` 반영 금지.

## 참석 / 맥락
- 팀 **망상궤도**: 김주용(리더·제품방향·리서치) · 이승보(개발·시스템 설계) · 김민주(디자인·UX) · 최영욱(자료 지원).
- 각자 UX/UI·개발 진척 공유 후, **제품 정의·앱 설계·발표 전략**을 구체화한 스프린트 회의. (직전 [[회의록-2026-06-30-아이디어회의]]에서 이어짐)

## 핵심 요약 (3줄)
1. **키보드-퍼스트 UX 확정** — 원 슬라이드 업무보드, 스페이스/방향키만으로 검토·수락(오승인 방지 위해 **스페이스 2회 승인**). 핵심 메시지 "**일은 에이전트가, 사람은 눈만 움직인다**".
2. **역할별 대시보드 + 로그인 라우팅** — RM·기업여신·전세보호 등 역할별 대시보드, 로그인 시 본인 역할 화면만. 에이전트 = 담당자 노하우 스킬 + MCP + 은행 DB + 국가 오픈 API + LLM, **최종 승인·책임은 사람**, **고객 PII 직접 미취급**.
3. **발표 전략** — 지역 공감 위해 **김민주 발표**(광주 연고), 시연 히어로 = **전주 카페 대출 심사**(예선 사례 계승). "MVP 실동작 + 산출물 신뢰 → 상위권".

## 공유·결정
1. **디자인·UX 1차 완료(김민주)**: JB 웹 분석 → 디자인 시스템(컬러 토큰·폰트·페이싱·레디우스·마진) + 컴포넌트 + 상태값 컬러·콘솔 레이아웃. **UX 라이팅** 전면 교체("강소 금융그룹"·고객 업무 지원 톤).
2. **개발 진척(이승보)**: 어제 UI 흐름 다듬기 + **비은행(JB우리캐피탈) 대시보드 병행**. Claude API 연결 후 Fable 5로 화면 퀄리티↑.
3. **키보드-퍼스트 UX(확정 방향)**: 업무보드 = **원 슬라이드 페이지**. 스페이스/방향키로 순차 검토·수락 → 마우스 최소화(업무량 많은 담당자 손목 부담↓ = **차별화 포인트**). **오승인 방지 = 스페이스 2회**(1회=내용 열람, 스크롤 후 2회=승인). 슬로건/키워드화(원클릭 대체).
4. **역할별 대시보드 + 로그인(개발설계, 이승보)**: RM·기업여신 담당자·전세보호 담당자 등 역할별 대시보드. 배포 시 **로그인 → 역할군 아이디 → 본인 역할 대시보드만 표시**. 좌측 사이드바에서 담당자별 전환. 계열사(전북/광주/캐피탈) 각각 내부에서 동일 구조.
5. **에이전트 구조(제품 방향, 김주용)**: 담당자 R&R 중 에이전트가 할 수 있는 업무를 세팅. 구성 = 담당자 **노하우 스킬(암묵지)** + 기업 업무 스킬 + **MCP**(팀 일정·금융 MCP·정책 문서) + **은행 내부 DB** + **국가 오픈 API**(국토부 실거래가·전세·정책 규정) + **LLM**. 위험성 분석 반복. **최종 승인·책임 = 사람(담당자)**. **고객 개인정보는 에이전트가 직접 다루지 않음**(해킹·유출 방지). "아는 것 많은 RM의 비서."
6. **케이스/고객 메모리 제안**: 케이스·고객별 메모리(통화내역·상담·시계열 데이터) 축적 → 담당자 확인·판단 지원.
7. **신뢰·판단 해법**: 금융권 신뢰 장벽 → "데이터 제공 시 기존 담당자 노하우·판단을 AI 학습 + 실시간 데이터 + 등록 스킬 결합 → 판단 신뢰도↑".
8. **발표 전략**: 프로토타입 시연 = 팀원 개인정보로 연기(전주/광주 연고). 히어로 = **전주 카페 창업 대출 심사**(담당자→심사위원→고객 시선 연결). 발표자 = **김민주**(지역 공감대). 보이스피싱(노인 송금 설문→위험점수→차단)도 시연 후보.
9. **평가·범위 인식**: 평가 각 20점(혁신성·완결성·MVP·사업화). "**MVP 실동작 + 산출물 신뢰 → 3등 이상 가능**"(라이브코딩 paperclip 기반 설계 = 경쟁 우위). 도메인 지식 부족 인정 → 백엔드 전체·시장분석 과열 경계, **MVP+발표 집중**. 자유주제(그룹 확장성) 강조.
