# 재설계-방향

> 대외비 — 6/29 공식발표 전 비공개.

## 핵심 사실

**결정**: 현재 정적 MVP UI 폐기 → 역엔지니어링 재설계

**접근법 A (채택 방향)**
- 백엔드: Node.js + TypeScript
- 실행 환경: 로컬 + 클라우드 혼합
- Rule Engine + LLM 토글 방식
- 현재 상태: 미구현 (설계 단계)

**배경**
- 기존 정적 프로토타입은 에이전트 흐름 시연에 한계
- 심사 기준상 실제 작동 루프 시연이 필수 평가 항목
- paperclip 레퍼런스 기반 라우터/데이터 구조 재설계
