# 03_제품

실행 가능한 제품 화면, PRD, 기능 명세를 관리하는 공간입니다.

## 제품명과 정의

`JB LocalGuard OS`

지역 금융 고객의 위험 신호를 수집하고, Agent가 스킬을 장착해 분석하며, 사람 승인 전까지 외부 행동을 통제하는 금융 Agent 운영 콘솔입니다.

## 제품 컨셉

- Case: 고객/매물/사업자 위험을 하나의 작업 단위로 묶습니다.
- AgentRun: Agent가 케이스 맥락과 스킬을 장착해 실행한 기록입니다.
- Skill Registry: Agent별 처리 능력을 독립 스킬로 관리합니다.
- Approval Gate: 고객-facing 행동 전 사람 승인 단계를 둡니다.
- Audit Ledger: 판단, 실행, 승인, 반려의 근거를 남깁니다.

## 주요 화면과 역할

| 화면 | 역할 |
| --- | --- |
| 대시보드 | 케이스 상태, 긴급 알림, 운영 지표를 빠르게 확인 |
| 알림함 | 승인 대기, 차단, SLA 임박 항목 확인 |
| 케이스 | 위험 케이스 목록과 상세 패널 연결 |
| 승인 | RM/준법 사람이 승인하거나 반려할 조치 관리 |
| 실행 이력 | AgentRun 실행 로그와 진행 상태 확인 |
| 전세 보호 | 전세사기 위험 신호, 자산노출, 계약 체크리스트, 은행 연계 흐름 |
| 에이전트 팀 | 운영 지휘, 위험/금융 판단, 전세 보호, 준법 통제, 고객/은행 연계로 Agent 분류 |
| 조직도 | 사람 승인자와 AI Agent의 보고 체계 시각화 |
| 자동화 | 정기 실행 루틴과 상태 점검 |
| 운영 목표 | triage 시간, 근거 추적성, 승인 안전성, 전세 안전계약 목표 |
| 스킬 | Agent에 장착되는 금융/리스크/계약/준법 스킬 |
| 활동 이력 | 승인, 실행, 상태 변경, 상위 검토 기록 |
| 비용 | Agent별 API 예산과 사용률 |
| 설정 | 조직 프로필, 승인 정책, 외부 연동 어댑터 |

## MVP에서 실제로 확인할 수 있는 동작

- 케이스 선택 시 오른쪽 상세 패널 갱신
- AgentRun 생성과 실행 이력 추가
- 승인/반려 버튼을 통한 상태 변경
- 감사 로그 추가
- 에이전트/스킬/기능 카드 클릭 시 상세 패널 연결
- 오른쪽 상세 패널 on/off
- 상세 패널 내부 항목 접기/펼치기
- 업무 범주별 에이전트 팀 분류
- 현재 부족한 부분 패널에서 실데이터, 점수 산식, 권한/보안, 품질 검증 과제 확인
