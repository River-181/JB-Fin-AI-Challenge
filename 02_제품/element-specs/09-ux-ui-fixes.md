# 09 · UX/UI 오류 수리 + 상세 패널 스크롤

> 모듈: styles.css + 렌더 가드 · 데모 인상 품질

## 확인된 오류 / 수리 대상
1. **상세 패널 스크롤 불가 (사용자 보고)**
   - 원인: `.context-panel > section.panel:nth-of-type(2~4)`에 하드 `max-height`(206/132/122px) + 내부 `overflow` 캡(approval-gates 66px 등). 내용이 잘리고 스크롤도 막힘.
   - 수리: nth-of-type max-height 캡 제거/완화, 패널 내부는 자연 높이 + 컨테이너 단위 스크롤. context-panel 자체 `overflow-y:auto`.
2. **요약-only 한계**: 클릭 시 우측 요약만 → 전용 케이스 상세 뷰로 진입(01).
3. **죽은/잘린 상태 점검**: 빈 상태 메시지, 비활성 버튼 사유 표기, 긴 텍스트 줄바꿈.
4. **대비/계층**: (이전 패스 반영) 저대비 텍스트·보조 텍스트 weight 점검 유지.
5. **거버넌스/산출물 등 신규 섹션의 모바일·태블릿 반응형** 확인(기존 e2e 뷰포트 테스트 통과 유지).
